Welcome to qpwoeirut.com!

Here you can find a selection of fun arcade games to play whenever you want. All of the projects are written using JavaScript and HTML, so there is no need for Adobe Flash Player. If you have any questions or comments for us, see the Contact Us page or visit the project repository on Github. Feel free to submit an issue or concern as well.


Our current catalog of games consists of:

  • Minesweeper

  • Snake

  • Conway's Game of Life

  • Pong

  • Hangman

  • Breakout


In the future, we plan to work on:

  • Quadratic Solver
