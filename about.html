About the Site

qpwoeirut.com was created by Jonathan Uhler and Stanley. We are two high school students who are interested in computer science and programming. The idea for this website came up during the COVID-19 pandemic, since we needed something to do during lockdown.

We decided to look into JavaScript and HTML and began to work on an arcade-themed website. Our site offers a selection of simple games to play whenever you want. The site is also static, so you can download the games to play offline by cloning the repository on GitHub.

All of our work is written in JavaScript, HTML, CSS, and XML using Visual Studio Code. The project repository can be found on GitHub here.

How To Play Offline

Since this entire website is static, it can be downloaded and everything should still work. To download the website you can go to our repository on GitHub and clone it. Then to play just open up the website file in your web browser.
